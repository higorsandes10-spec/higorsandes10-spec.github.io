Seção 2 - Ranking de tecnologias

  1. SQL - Gerencia o armazenamento e a consulta de dados em bancos de dados.

  2. React – Facilita a criação de interfaces modernas e dinâmicas.

  3. JavaScript – Adiciona interatividade e funcionalidades às páginas.

  4. CSS3 – Define a aparência e o estilo visual dos sites.

  5. HTML5 – Estrutura e organiza o conteúdo das páginas web.

Seção 3 - Glóssario de termos Técnicos

API
Interface de Programação de Aplicações.
Conjunto de rotinas e padrões para acesso a um software.
Navegador
Browser
Software utilizado para acessar a internet.
URL
Endereço web.
DNS
Sistema que converte nomes de domínio em endereços IP.
HTTP
Protocolo de transferência de hipertexto.
DOM
Modelo de Objeto de Documento.
